# Branches 페이지 이미지 파일 안내

본점 지점 안내 탭에서 사용되는 SNS 아이콘 이미지들을 이 디렉토리에 저장해야 합니다.

## 필요한 이미지 파일

다음 파일들을 `/public/images/branches/icons/` 디렉토리에 저장하세요:

1. **naver-blog.svg** - 네이버 블로그 아이콘 (20x20px 권장)
2. **youtube.svg** - YouTube 아이콘 (20x20px 권장)
3. **instagram.svg** - Instagram 아이콘 (20x20px 권장)
4. **website.svg** - 웹사이트 아이콘 (20x20px 권장)

## 이미지 스펙

- **형식**: SVG (벡터 이미지 권장) 또는 PNG
- **크기**: 20x20px
- **색상**: 흰색 또는 투명 배경의 아이콘 (다크 테마에 맞춤)
- **스타일**: 단순하고 깔끔한 라인 아이콘

## 참고

이미지 파일이 없어도 페이지는 정상적으로 작동하지만, SNS 링크 아이콘이 표시되지 않습니다.
